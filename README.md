# E COMMERCE-TEKNIK-INFORMATIKA-3A

Proyek ini merupakan pengembangan **website e-commerce fashion** dengan fokus pada pengelolaan produk serta alur belanja end-to-end. Proyek ini dibuat untuk keperluan **tugas database teknik informatika 3A** dan pengembangan sistem berbasis web.

---

## Tujuan Proyek

- Membangun website e-commerce fashion berbasis web
- Mengimplementasikan alur belanja lengkap (browse → checkout)
- Mengelola produk fashion dengan varian dan stok
- Menerapkan konsep basis data relasional dalam sistem e-commerce
